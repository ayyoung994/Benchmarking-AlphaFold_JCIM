`Pocket_Consensus` denotes the consensus binding-site residue set shared across multiple receptor models and used as a common reference pocket for local JmjC-domain analysis.

`Pocket_RMSD` was calculated for the `Pocket_JmjC_CA` index group using the pre-aligned core-fitted trajectory (`md_0_60_fit_core_4.xtc`). No additional fitting was applied during RMSD calculation (`-fit none`).

`Pocket radius of gyration (Rg)` Pocket Rg was calculated for the `Pocket_JmjC_BB` index group using the centered trajectory (`md_0_60_center_4.xtc`).

For pocket-level dynamics, RMSD was evaluated using `Pocket_JmjC_CA`, whereas radius of gyration was evaluated using `Pocket_JmjC_BB` to monitor local backbone compactness of the consensus pocket region.
